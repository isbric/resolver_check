# resolver_check

Simple dnspython wrapper for some automated testing


## Install
Use of virtualenv is recommended
```bash
 pip install . -U
```
or
```bash
 pip install -U -e git+https://{{ url }}/resolver_check.git@v0.0.1.rc1#egg=resolver_check-v0.0.1.rc1
```

## Use
```bash
resolver_check -h
```
see exampletests.yaml
